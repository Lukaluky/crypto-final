## Registration

1. Open the website  
2. Enter a username and password  
3. Save the displayed TOTP secret  
4. Add it to Google Authenticator  

---

## Login

1. Enter username  
2. Enter password  
3. Enter TOTP code from Google Authenticator  
4. Access dashboard upon successful login  

---

## Secure Messaging

1. Enter the recipient's username  
2. Type your message  
3. Message is encrypted and sent  
4. Recipient sees the decrypted message  

---

## File Encryption

**Encrypt a File:**  
1. Select file  
2. Click **Encrypt**  
3. Download encrypted file  

**Decrypt a File:**  
1. Upload encrypted file  
2. Click **Decrypt**  
3. Original file is restored  

---

## Audit Trail

1. Click **Audit**  
2. View all logged security events  
3. Logs are immutable and cannot be modified
